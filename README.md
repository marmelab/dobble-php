# dobble-php
Command-line card generator for dobble-like game

# Installation

Run the following command

```bash
make install
```

# Use

Pending the final .phar file, you can use symfony command :

```bash
php src/application.php dobble:run -h
```

# Unit tests

```bash
make test```
